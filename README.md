# odin-recipes

Intro
A basic recipe website consisting of a main index page with links to several recipes. 

Objective
The purpose of this project is to showcase skills learnt through the Odin Project, as it relates to:
    - Downloading repository from Github to local machine using VS Code terminal and Git
    - Customizing a home page and individual recipe pages with html elements
    - Updating Github repository using VS Code terminal
    - Creating meaningful messages for Github commits
    - Pushing updates to the main branch of Github repository
