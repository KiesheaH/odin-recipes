# odin-recipes
Comfort Cravings is a website that offers users four comfort food recipes. The website was created as a project submission for The Odin Project - CSS Foundations Course - Odin-recipe. 

This website showcases skills learnt, thus far, as it relates to:
- Downloading repository from Github to local machine using VS Code terminal and Git
- Customizing a home page and individual recipe pages with html elements and css styles
- Updating Github repository using VS Code terminal
- Creating meaningful messages for Github commits
- Pushing updates to the main branch of Github repository
